# All-in-One Calculator App

## Overview
A feature-rich calculator app built with HTML, CSS, and JavaScript. It includes **scientific, financial, and loan calculations**, a **calculation history**, and **dark mode**. Fully responsive and easy to use.

## Features
- Scientific calculations (trigonometry, logarithms, exponents)
- Financial & loan calculations (interest rates, loan payments)
- Calculation history
- Dark mode
- Responsive design (desktop & mobile)

## Why Use It
Perfect for students, professionals, or businesses who need a versatile calculator ready to use or integrate into web projects. Fully customizable and deployable.
